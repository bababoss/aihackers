üöÄ Running ComfyUI with Manager Access

Start ComfyUI with the following command to enable the manager and make it accessible:

python main.py --enable-manager --listen 0.0.0.0 --port 8000

For Local Machine: Access ComfyUI at http://localhost:8000

üîó Remote Access via SSH Tunneling (Azure GPU VM)

To securely access ComfyUI running on a remote Azure GPU VM, use SSH tunneling:

ssh -L 8000:localhost:8000 -i amdgpu_key.pem azureuser@xx.xx.xx.xx.ip

How it works:

  • -L 8000:localhost:8000 - Maps local port 8000 to remote port 8000
  • -i amdgpu_key.pem - SSH private key for authentication
  • azureuser@xx.xx.xx.xx.ip - Replace with actual Azure VM IP address

After SSH tunnel is established: Access ComfyUI at http://localhost:8000 from your local machine.

üí° Pro Tip: This setup allows you to leverage powerful Azure GPU VMs (like GPU-optimized instances) while controlling ComfyUI from your local browser. Ideal for running computationally intensive video generation without maxing out your local hardware.

üöÄ Quick Start Guide

Getting started with ComfyUI-LTXVideo is surprisingly simple. Follow these steps:

  1. Install ComfyUI

    Download and install ComfyUI from comfy.org if you haven't already.

  2. Open ComfyUI

    Launch ComfyUI in your web browser (typically at http://localhost:8188).

  3. Access the Manager

    Click the Manager button in the UI (or press Ctrl+M).

  4. Install Custom Nodes

    Select "Install Custom Nodes" from the Manager menu.

  5. Search for LTXVideo

    Search for "LTXVideo" in the custom nodes list.

  6. Click Install

    Click the install button and wait for the installation to complete.

  7. Restart ComfyUI

    Restart ComfyUI for the changes to take effect.

  8. Start Creating!

    LTXVideo nodes will now appear in your node menu under "LTXVideo" category. Required models will download automatically on first use.

‚úì Models Download Automatically

You don't need to manually download models. ComfyUI will fetch everything on first use. Just make sure you have enough disk space!

üîß Troubleshooting Common Issues

‚ùå Issue: "Out of Memory" Errors

Solution: Switch to the distilled model, enable FP8 quantization, or use low VRAM loaders from the repository.

‚ùå Issue: Very Slow Generation

Solution: Use the distilled model variant, reduce video resolution, check that GPU is being utilized (not CPU), and ensure sufficient vRAM.

‚ùå Issue: Models Not Downloading

Solution: Ensure you have 100GB+ free space, check internet connection, and verify HuggingFace model access.

‚ùå Issue: ComfyUI Won't Load LTXVideo Nodes

Solution: Restart ComfyUI after installation, check for error logs in the terminal, and verify Python dependencies are installed.
