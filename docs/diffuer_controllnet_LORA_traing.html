Section 01

What is PEFT & LoRA?

Parameter-Efficient Fine-Tuning (PEFT) is a family of techniques that adapt large pre-trained models to new tasks by updating only a tiny fraction of their parameters — keeping the rest frozen. LoRA (Low-Rank Adaptation) is the dominant PEFT method for diffusion models: instead of updating the full weight matrix W, it learns a low-rank factorization ΔW = B·A whose rank r ≪ d makes it 10–1000× cheaper to train than full fine-tuning.

🧊
Frozen Backbone
The pre-trained model (SD 1.5, SDXL, DiT) never receives gradient updates — all its learned priors are preserved exactly.
🧮
Low-Rank Matrices
Two small matrices B ∈ ℝd×r and A ∈ ℝr×k replace the full ΔW ∈ ℝd×k. With r=4, params drop from d·k to (d+k)·r.
⚡
Minimal VRAM
LoRA adapters for a full SD U-Net weigh ~3–80 MB vs 4 GB for the full model. Fine-tune on a single consumer GPU (8–16 GB).
🔀
Merges at Inference
At inference time, ΔW = B·A is merged back into W′ = W + α·ΔW — zero extra latency compared to the base model.
🏗
Architecture Agnostic
Works on any linear layer: attention Q/K/V/O projections, feed-forward layers, cross-attention — even ControlNet branches.
🔌
Composable Adapters
Multiple LoRA adapters (style + defect type + surface material) can be loaded simultaneously with independent scaling weights α.
LoRA — Core Math
Standard linear layer: h = W₀ · x
LoRA modification: h = W₀ · x + (B · A) · x · α/r
where: W₀ ∈ ℝd×k (frozen), B ∈ ℝd×r (init 𝒩(0,σ)), A ∈ ℝr×k (init 0)
Trainable params per layer: (d + k) × r vs d × k full
Example (SD attn proj, d=k=768, r=4): 0.1% of original layer params

LoRA Mechanics — Animated Weight Decomposition

Every attention layer in a U-Net contains projection matrices (Q, K, V, Out). LoRA adds a parallel low-rank bypass: a tiny B×A path whose output is scaled by α/r and added to the frozen W₀ output. The animation below shows how a rank-4 update flows through a 768×768 attention projection at training time.

🧮 Canvas A — LoRA Weight Matrix Decomposition (Live Animation)
Why initialise A with 𝒩(0,σ) and B with zeros?
At the start of training, B·A = 0 so the LoRA adapter contributes nothing — the model behaves identically to the frozen base. This means the very first gradient step is well-conditioned and does not destroy the base model's outputs. Only A is random, ensuring B·A is not stuck at a saddle point.
Gradient Flow — what gets updated
Forward: h = W₀·x + (B·A)·x · (α/r)
Loss: ℒ = MSE( ε_θ(z_t, t, c) , ε )
∂ℒ/∂W₀ = 0     (frozen, no grad accumulation → saves VRAM)
∂ℒ/∂B = (∂ℒ/∂h) · (Ax)ᵀ · (α/r)
∂ℒ/∂A = Bᵀ · (∂ℒ/∂h) · xᵀ · (α/r)
Optimizer updates only (A, B) — 2×r×(d+k) floats per layer

⚙️ Rank Selection — Effect on Params & Quality (SD 1.5 U-Net, d=k=768)

Rank r = 1
0.026%
~1,536 params/layer. Minimal capacity — good for colour/tone shift only.
Rank r = 4
0.10%
~6,144 params/layer. Sweet spot for style & surface texture adaptation.
Rank r = 16
0.42%
~24,576 params/layer. Handles complex defect topology and structure changes.
Rank r = 64
1.67%
~98,304 params/layer. Near full fine-tune quality; use when data is abundant.
α (scale)
1 – 2×r
Controls adapter strength. α=r gives unit scale; α=2r doubles the update magnitude.
Target modules
Q K V O
Apply LoRA to all attention projections + optionally feed-forward for max quality.
Python — Attach LoRA to a Diffusers UNet (HuggingFace PEFT)
from diffusers import UNet2DConditionModel
from peft import LoraConfig, get_peft_model, TaskType

# 1. Load frozen base UNet
unet = UNet2DConditionModel.from_pretrained(
    "runwayml/stable-diffusion-v1-5", subfolder="unet"
)

# 2. Define LoRA config — targets all attention projections
lora_config = LoraConfig(
    r=4,                          # rank
    lora_alpha=4,                  # scale α = r  → unit scale
    target_modules=["to_q", "to_k", "to_v", "to_out.0"],
    lora_dropout=0.05,
    bias="none",
)

# 3. Wrap UNet — only LoRA (A, B) matrices are trainable
unet = get_peft_model(unet, lora_config)
unet.print_trainable_parameters()
# → trainable params: 797,696 || all params: 859,520,964 || 0.09%

# 4. Freeze everything except LoRA
for name, param in unet.named_parameters():
    if "lora_" not in name:
        param.requires_grad = False

# 5. Save only the adapter (not the full model)
unet.save_pretrained("./lora_wall_defect_r4")   # ~3 MB file

# 6. Load and merge at inference
from peft import PeftModel
unet_base = UNet2DConditionModel.from_pretrained(
    "runwayml/stable-diffusion-v1-5", subfolder="unet"
)
unet_lora = PeftModel.from_pretrained(unet_base, "./lora_wall_defect_r4")
unet_merged = unet_lora.merge_and_unload()   # W′ = W₀ + BA, zero latency

Combined Architecture — Diffusion + ControlNet + LoRA

The three components stack as independent, complementary layers on top of the same frozen SD backbone. LoRA adapters sit inside the attention projections of both the frozen U-Net AND the ControlNet branch. Animated dots show signal flow at inference time.

🏗 Canvas C — Full Stacked Architecture (Frozen / ControlNet / LoRA layers animated)
🧊 Frozen SD Backbone
Pre-trained U-Net encoder + bottleneck + decoder. Weights never change. Retains all photorealistic texture priors learned on LAION-5B. Grey blocks in the animation.
🎛 ControlNet Branch
Copied & trainable encoder + zero-convolutions. Conditions on spatial hint (edge/mask/depth). Red blocks — trained separately, injected into frozen decoder skip connections.
✨ LoRA Adapters
Gold pulsing overlays on every attention layer (Q/K/V/O). Only B×A matrices are trained. Applied to both SD U-Net attn AND ControlNet attn for domain-specific surface appearance.

Python — Dataset Class: Wall / Mask / Prompt Triplets
import torch
from torch.utils.data import Dataset
from pathlib import Path
from PIL import Image
import torchvision.transforms as T
import json, random

class WallDefectDataset(Dataset):
    """
    Directory structure:
      data/
        images/   *.jpg   — clean wall photos
        masks/    *.png   — binary defect masks (0=clean, 255=defect)
        edges/    *.png   — Canny/HED edge maps
        depth/    *.png   — MiDaS depth maps
        captions/ *.json  — {"short":..., "medium":..., "detailed":...}
    """
    def __init__(self, root, resolution=512, augment=True):
        self.root       = Path(root)
        self.ids        = sorted(p.stem for p in (self.root/"images").glob("*.jpg"))
        self.resolution = resolution
        self.augment    = augment
        self.tfm = T.Compose([
            T.Resize((resolution, resolution), interpolation=T.InterpolationMode.BILINEAR),
            T.ToTensor(),
            T.Normalize([0.5], [0.5]),   # → [-1, 1]
        ])
        self.mask_tfm = T.Compose([
            T.Resize((resolution, resolution), interpolation=T.InterpolationMode.NEAREST),
            T.ToTensor(),
        ])

    def __len__(self):
        return len(self.ids)

    def __getitem__(self, idx):
        stem = self.ids[idx]

        image = Image.open(self.root/"images"/f"{stem}.jpg").convert("RGB")
        mask  = Image.open(self.root/"masks" /f"{stem}.png").convert("L")
        edge  = Image.open(self.root/"edges" /f"{stem}.png").convert("RGB")

        # Random horizontal flip augmentation
        if self.augment and random.random() > 0.5:
            image = image.transpose(Image.FLIP_LEFT_RIGHT)
            mask  = mask.transpose(Image.FLIP_LEFT_RIGHT)
            edge  = edge.transpose(Image.FLIP_LEFT_RIGHT)

        captions = json.loads((self.root/"captions"/f"{stem}.json").read_text())
        prompt   = random.choice([captions["short"], captions["medium"], captions["detailed"]])

        return {
            "pixel_values":     self.tfm(image),       # (3, H, W)  [-1,1]
            "conditioning_image": self.tfm(edge),        # ControlNet hint
            "mask":              self.mask_tfm(mask),   # (1, H, W)  [0,1]
            "prompt":            prompt,
        }

# Usage
dataset = WallDefectDataset("./data", resolution=512)
loader  = torch.utils.data.DataLoader(dataset, batch_size=4, shuffle=True, num_workers=4)
print(f"Dataset: {len(dataset)} samples, {len(loader)} batches")

📋 Training Phases

Phase 1 — Warm-up: ControlNet Only (5k–10k steps)

Train only the ControlNet branch and Zero Convolutions. UNet LoRA adapters are frozen. Batch size 4, LR 1e-4, DDPM scheduler. Goal: learn to extract spatial features from hint images.

Phase 2 — Joint Training: ControlNet + LoRA (20k–50k steps)

Enable LoRA adapter gradients alongside ControlNet. The LoRA adapters in both the frozen UNet and ControlNet branch are unfrozen. LR decayed to 2e-5. ControlNet conditioning_scale annealed from 0.5 → 1.0 over first 10k steps to avoid dominance.

Phase 3 — Domain Fine-Tune: LoRA Only (2k–5k steps)

Freeze ControlNet branch completely. Continue training only LoRA adapters on target-domain images (e.g. specific building material, lighting condition). LR 5e-6. This "last-mile" phase adapts appearance without disturbing spatial conditioning.

Phase 4 — Evaluation & Checkpoint (every 500 steps)

Generate 64 synthetic samples with held-out masks. Compute FID against real validation images. Log FID / SSIM / LPIPS to WandB. Save checkpoint if FID improves. Run downstream defect-detector (YOLOv8) on synthetic augmented set; track mAP@0.5.

Complete Runnable Code

Three production-ready code samples: the full training script that orchestrates all three components, an inference pipeline that loads checkpoints and generates controlled defect images, and helper utilities for augmentation and evaluation.

Python — Full Training Script (Diffusers + ControlNet + LoRA + accelerate)
#!/usr/bin/env python3
# train_controlnet_lora.py — End-to-end wall defect synthesis training
import torch, argparse, yaml, wandb
from pathlib import Path
from accelerate import Accelerator
from diffusers import (
    AutoencoderKL, UNet2DConditionModel, ControlNetModel,
    DDPMScheduler, StableDiffusionControlNetPipeline
)
from transformers import CLIPTextModel, CLIPTokenizer
from peft import LoraConfig, get_peft_model
from dataset import WallDefectDataset   # from Section 4

def main(cfg):
    accelerator = Accelerator(mixed_precision=cfg["mixed_precision"], log_with="wandb")
    accelerator.init_trackers("wall-defect-synthesis")

    # ── Load models ──────────────────────────────────────────
    vae         = AutoencoderKL.from_pretrained(cfg["pretrained_model"], subfolder="vae")
    text_enc    = CLIPTextModel.from_pretrained(cfg["pretrained_model"], subfolder="text_encoder")
    tokenizer   = CLIPTokenizer.from_pretrained(cfg["pretrained_model"], subfolder="tokenizer")
    unet        = UNet2DConditionModel.from_pretrained(cfg["pretrained_model"], subfolder="unet")
    controlnet  = ControlNetModel.from_pretrained(cfg["controlnet_model"])
    noise_sched = DDPMScheduler.from_pretrained(cfg["pretrained_model"], subfolder="scheduler")

    # ── Freeze what should not train ─────────────────────────
    vae.requires_grad_(False); text_enc.requires_grad_(False)

    # ── Attach LoRA to UNet attention layers ─────────────────
    lora_cfg = LoraConfig(
        r=cfg["lora_rank"], lora_alpha=cfg["lora_alpha"],
        target_modules=cfg["target_modules"], lora_dropout=0.05, bias="none"
    )
    unet = get_peft_model(unet, lora_cfg)

    # ── Dataset ───────────────────────────────────────────────
    dataset = WallDefectDataset(cfg["data_root"], resolution=cfg["resolution"])
    loader  = torch.utils.data.DataLoader(
        dataset, batch_size=cfg["train_batch_size"], shuffle=True, num_workers=4
    )

    # ── Optimizer — only LoRA + ControlNet params ─────────────
    params = list(filter(lambda p: p.requires_grad,
                         list(unet.parameters()) + list(controlnet.parameters())))
    optim  = torch.optim.AdamW(params, lr=cfg["learning_rate"], weight_decay=1e-2)

    unet, controlnet, optim, loader = accelerator.prepare(unet, controlnet, optim, loader)
    vae = vae.to(accelerator.device, dtype=torch.float16)
    text_enc = text_enc.to(accelerator.device, dtype=torch.float16)

    global_step = 0
    for epoch in range(9999):
        for batch in loader:
            with accelerator.accumulate(unet, controlnet):

                # Encode images to latent space
                with torch.no_grad():
                    latents = vae.encode(
                        batch["pixel_values"].to(dtype=torch.float16)
                    ).latent_dist.sample() * 0.18215

                # Sample noise and timestep
                noise  = torch.randn_like(latents)
                bsz    = latents.shape[0]
                steps  = torch.randint(0, noise_sched.config.num_train_timesteps,
                                       (bsz,), device=latents.device).long()
                noisy_latents = noise_sched.add_noise(latents, noise, steps)

                # Encode text prompts
                with torch.no_grad():
                    tokens = tokenizer(batch["prompt"], padding="max_length",
                                       max_length=77, return_tensors="pt",
                                       truncation=True).input_ids.to(latents.device)
                    enc_hidden = text_enc(tokens)[0].to(dtype=torch.float16)

                # ControlNet forward pass
                cn_out = controlnet(
                    noisy_latents, steps, enc_hidden,
                    controlnet_cond=batch["conditioning_image"].to(dtype=torch.float16),
                    return_dict=False
                )
                down_blk_res, mid_blk_res = cn_out[0], cn_out[1]

                # U-Net + LoRA forward — predict noise
                noise_pred = unet(
                    noisy_latents, steps, enc_hidden,
                    down_block_additional_residuals=down_blk_res,
                    mid_block_additional_residual=mid_blk_res,
                ).sample

                # MSE loss against original noise
                loss = torch.nn.functional.mse_loss(
                    noise_pred.float(), noise.float(), reduction="mean"
                )
                accelerator.backward(loss)
                if accelerator.sync_gradients:
                    accelerator.clip_grad_norm_(params, 1.0)
                optim.step(); optim.zero_grad()

            global_step += 1
            if global_step % 50 == 0:
                accelerator.log({"loss": loss.detach().item()}, step=global_step)

            if global_step >= cfg["max_train_steps"]: break
        if global_step >= cfg["max_train_steps"]: break

    # Save LoRA adapter + ControlNet weights
    unet.save_pretrained(Path(cfg["output_dir"])/"lora_unet")
    controlnet.save_pretrained(Path(cfg["output_dir"])/"controlnet")
    accelerator.end_training()

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--config", required=True)
    args = parser.parse_args()
    with open(args.config) as f:
        cfg = yaml.safe_load(f)
    main(cfg)
Python — Inference: Load LoRA + ControlNet → Generate Defect
from diffusers import StableDiffusionControlNetPipeline, ControlNetModel, UniPCMultistepScheduler
from peft import PeftModel
import torch
from PIL import Image

CKPT_DIR = "./checkpoints/wall_lora_cn"
BASE_MODEL = "runwayml/stable-diffusion-v1-5"

# 1. Load fine-tuned ControlNet
controlnet = ControlNetModel.from_pretrained(
    f"{CKPT_DIR}/controlnet", torch_dtype=torch.float16
)

# 2. Build pipeline
pipe = StableDiffusionControlNetPipeline.from_pretrained(
    BASE_MODEL, controlnet=controlnet, torch_dtype=torch.float16
).to("cuda")

# 3. Load LoRA weights into UNet
pipe.unet = PeftModel.from_pretrained(
    pipe.unet, f"{CKPT_DIR}/lora_unet"
).merge_and_unload()    # W′ = W₀ + BA — zero inference overhead

# 4. Use fast scheduler
pipe.scheduler = UniPCMultistepScheduler.from_config(pipe.scheduler.config)
pipe.enable_xformers_memory_efficient_attention()

# 5. Generate synthetic defect image
hint  = Image.open("crack_mask.png").convert("RGB")   # edge / mask condition

result = pipe(
    prompt="concrete wall with a photorealistic hairline crack, construction site, HDR, 4k",
    negative_prompt="cartoon, painting, blurry, oversaturated, text",
    image=hint,
    controlnet_conditioning_scale=0.9,
    guidance_scale=7.5,
    num_inference_steps=25,       # UniPC is fast — 25 steps suffice
    generator=torch.Generator("cuda").manual_seed(42)
).images[0]

result.save("synthetic_crack_output.jpg")
print("✅ Saved — label = crack_mask.png (free annotation!)")
